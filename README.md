# 🛒 ショッピング電卓アプリ（Shop Calculator Lite）

スマートフォン向けのシンプルな買い物支援アプリ。  
商品を登録して、タップ＋個数入力で合計金額を即計算できます。

👉 サーバー不要・完全ローカル動作

---

## ✨ 特徴

- 📱 スマホ特化UI（片手操作想定）
- ⚡ `index.html` を開くだけで動作
- 💾 LocalStorageでデータ保存（オフラインOK）
- 🧮 商品がない場合は「普通の電卓」として使用可能
- 📦 JSONで商品データのインポート／エクスポート対応

## 🚀 使い方

### ① 起動
index.html をブラウザで開くだけ

### ② 商品登録
- 右上「商品管理」から登録
- 名前・価格・カテゴリ（1〜5）を入力

### ③ 買い物操作
1. 商品をタップ
2. 個数入力
3. 自動でカートに追加

### ④ 精算
- 「清算」ボタンで一覧表示
- 合計金額を確認
- 確定でリセット

---

## 📊 データ仕様

### 商品データ（LocalStorage: `calc_products`）
```json
[
  { "id": "abc123", "name": "りんご", "price": 120, "category": 1 }
]
```

### カート（LocalStorage: `calc_cart`）
```json
[
  { "productId": "abc123", "quantity": 3 }
]
```
